# 🧹 YouTube Cleaner 🚫🎯

Минималистичное расширение для браузеров на базе Chromium (Yandex Browser, Chrome, Edge, Opera), которое **убирает отвлекающие элементы YouTube** и помогает сосредоточиться.

---

## ✨ Возможности

Расширение позволяет управлять интерфейсом YouTube через простые переключатели:

### 🔘 Shorts
Полностью скрывает все виды Shorts:
- Shorts в основной ленте  
- Shorts в поиске  
- Shorts-блоки (reels / shelf)  
- Видео с бейджем `SHORTS`  
- Пункт **Shorts** в боковом меню  
- Все ссылки вида `/shorts/...`

---

### 🏠 Главная страница (Home Feed)
- Полностью убирает рекомендованную ленту на главной  
- Оставляет только:
  - поиск  
  - шапку  
  - навигацию  

👉 Идеально для режима **“зашёл с конкретной целью”**

---

## 🧠 Как это работает

Расширение:
- внедряет CSS-стили на страницы YouTube
- использует современные селекторы (`:has()`)
- скрывает элементы по:
  - DOM-структуре
  - атрибутам (`overlay-style="SHORTS"`)
  - URL (`/shorts/...`)

💡 Всё происходит локально — без вмешательства в сеть или API.

---

## 📦 Установка (вручную)

Подходит для:
- Yandex Browser  
- Google Chrome  
- Microsoft Edge  
- Opera  

### Шаги

1. Скачайте репозиторий  
   - `Code` → `Download ZIP`
2. Распакуйте архив
3. Откройте страницу расширений:

browser://extensions

4. Включите **Режим разработчика**
5. Нажмите **Загрузить распакованное расширение**
6. Выберите папку проекта

---

## 🔘 Использование

1. Нажмите на иконку расширения  
2. Включите нужные переключатели:

- **Shorts** — скрывает короткие видео  
- **Home Feed** — убирает главную страницу  

Изменения применяются мгновенно ⚡

---

## 🔐 Конфиденциальность

- ❌ не собирает данные  
- ❌ не использует сторонние скрипты  
- ❌ не отправляет запросы  
- ✅ работает полностью локально  
- ✅ использует только `storage` для сохранения настроек
