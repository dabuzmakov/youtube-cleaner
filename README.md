# Hide YouTube Shorts 🚫📱

Минималистичное расширение для браузеров на базе Chromium (Yandex Browser, Chrome, Opera и др.), которое **полностью скрывает YouTube Shorts** со всех разделов сайта.

---

## ✨ Возможности

Расширение убирает **все виды Shorts**:

- Shorts в основной ленте YouTube  
- Shorts в результатах поиска (включая grid‑блоки)  
- Shorts, замаскированные под обычные видео (бейдж `SHORTS`)  
- Пункт **Shorts** в левом меню (обычном и свернутом)  
- Страницу `/shorts` и все ссылки на неё  

Работает через **чистый CSS + DOM‑селектора `:has()`**, без тяжёлого JavaScript.

---

## 🧠 Как это работает

Расширение:
- внедряет CSS‑стили на страницы YouTube
- определяет Shorts по:
  - `overlay-style="SHORTS"`
  - ссылкам `/shorts/...`
  - DOM‑структурам YouTube
- скрывает соответствующие элементы целиком

Никакие запросы, трекеры или аналитика **не используются**.

---

## 📦 Установка (вручную)

Подходит для:
- Yandex Browser  
- Chrome  
- Opera  
- Edge  

### Шаги

1. Скачайте репозиторий  
   - `Code` → `Download ZIP`
2. Распакуйте архив
3. Откройте страницу расширений:
   - Yandex / Chrome / Edge:  
     ```
     browser://extensions
     ```
4. Включите **Режим разработчика**
5. Нажмите **Загрузить распакованное расширение**
6. Выберите папку проекта

После этого иконка расширения появится справа вверху браузера.

---

## 🔘 Использование

- Нажмите на иконку расширения
- Включите или выключите тумблер **Hide Shorts**
- Изменения применяются мгновенно
